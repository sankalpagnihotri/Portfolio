001 : Portfolio

A Portfolio website is a collection of photographs, drawings, documents, etc. that you carry as an example of your work, especially when applying for a job.

This FullStack web application is developed using HTML, CSS, JavaScript and Bootstrap in frontend and FLASK (Framework of Python ) in the backend including database to store data.
